# Features
- add note
- delete note
- view note page && when it created on 
- bookmark feature
- google signin and signout
- Excellent search match
- image upload
- Loading bar
- Multi User Support
- Scroll to top button

# Technologies used
- Nextjs
- Tailwindcss
- firebase (for authentication only)
- sanity
- heroicons
